## AI Project Report Assistant

### Project Overview
Built an AI-driven automation system that converts raw project data (sprint boards, risk logs, spreadsheets) into polished weekly status reports and executive-ready summaries. <br>

The solution replaces manual reporting with consistent, insight-driven AI workflows, reducing reporting effort by up to 80%.<br>

### Key Activities & Execution 
-Created the project website using Lovable<br>
-Connected project data sources (Jira exports, risk logs, sprint data)from google drive for AI processing<br>
-Used LLMs (ChatGPT,NotebookLM) to generate weekly reports, RAG status, risks, and mitigation actions<br>
-Built executive summaries that translate technical data into business-focused insights<br>
-Generated presentation-ready slides for leadership updates<br>
-Automated report distribution and stakeholder notifications via Zapier and Gmail<br>
-Added an AI chatbot to answer stakeholder queries and provide instant project insights on-demand<br>
