# Inbox Inferno at Nexus Integrations

An AI-powered workflow built in n8n to process, analyze, and respond to high-volume inbox data using RAG (Retrieval-Augmented Generation) and evaluation datasets.

## Key Features
* Automated inbox processing using n8n workflows
* RAG-based response generation
* Integration with structured datasets (Google Sheets / Excel)
* Evaluation pipeline for response quality

## Workflow Details

  1. Input data (Inbox / uploaded files)<br>
  2. Preprocessing & formatting<br>
  3. Data ingestion into vector store<br>
  4. Retrieval using semantic search<br>
  5. AI response generation<br>
  6. Evaluation using test dataset<br>

## Tech Stack

* Vector Database – For semantic retrieval
* LLM (OpenAI) – Response generation
* Google Sheets / Excel – Data source
* RAG Pipeline – Context-aware AI responses
